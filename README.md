# 🧭 Matilda Mini Browser
Why M-Browser ?
In a world of heavy, resource-hungry browsers...you just found:

A lightweight desktop web browser. But...Remember, M-Browser wasn't created to replace Chrome; it was created to demonstrate that you can build a usable browser from Zero with Rust.

---

## ✨ Features

* 🌐 Basic web browsing
* 🔗 Address bar navigation
* ⚡ Lightweight and fast
* 🚫🥇 YouTube Without Ads, optimize web experience.
* 🌍🗣️✍️🏆 Ext translate sellect text into Vietnamese (All languages to Vietnamese Only)
* 🚀 Have no tab but new windows, increase performace, easy to End Task 
* 🖥️ Native Windows application

## ⚠️ Requirements

This application requires **Microsoft Edge WebView2 Runtime** to run.

If it is not installed on your system, the application may not start or display web content.

### 👉 Download WebView2 Runtime:

* Official download from Microsoft:
  https://developer.microsoft.com/en-us/microsoft-edge/webview2/

---

## 🚀 How to Use

1. Download the `.exe` file
2. Make sure WebView2 Runtime is installed
3. Run the application
4. Enter a URL in the address bar and start browsing
