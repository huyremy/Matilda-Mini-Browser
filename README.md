# 🧭 Matilda Mini Browser

A lightweight desktop web browser built with Rust, FLTK, and WebView.

---

## ✨ Features

* 🌐 Basic web browsing
* 🔗 Address bar navigation
* ⚡ Lightweight and fast
* 🖥️ Native Windows application

---

## 📦 Download

Download the latest here:
👉 [*(Download here)*](https://github.com/huyremy/Matilda-Mini-Browser/blob/main/mini_browser.exe)

---

## ⚠️ Requirements

This application requires **Microsoft Edge WebView2 Runtime** to run.

If it is not installed on your system, the application may not start or display web content.

### 👉 Download WebView2 Runtime:

* Official download from Microsoft:
  https://developer.microsoft.com/en-us/microsoft-edge/webview2/

---

## 🚀 How to Use

1. Download the `.exe` file
2. Make sure WebView2 Runtime is installed
3. Run the application
4. Enter a URL in the address bar and start browsing

---

## 🛠️ Built With

* Rust
* FLTK
* WebView (wv)

---

## 📌 Notes

* This is a minimal browser project for learning and experimentation
* Some modern web features may not be fully supported
